com.jeantessier.dependency.CycleDetectorFixture

Given the class-level graph:

    a.A --> b.B --> c.C
     ^               |
     |               |
     +---------------+

The cycle should print out as:

    a.A
        --> b.B
            --> c.C
                --> a.A

Setup

Create some nodes

create a.A class
create b.B class
create c.C class

Create some dependencies.

source a.A depends on b.B
source b.B depends on c.C
source c.C depends on a.A

Exercise

detect cycles
